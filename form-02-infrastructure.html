How to fill infrastructure data
All 31 unit types below are listed by default and start unfilled. For each: mark Available = Yes/No. If Yes, enter Total Count and the condition breakdown — Functional Count is then calculated automatically as Good + Satisfactory + Unsatisfactory. Reason fields only appear when required (e.g. Not Utilized > 0, Non-Functional > 0, Dangerous = Yes). Additional Units Required can be marked regardless of availability.
